Godot

Flat Palette

Made with Godot

Flat Palette is a program I’m making to pick out color palettes for games and modeling. It’s mainly for low poly 3D modeling, as I use that art style a lot in my games. Eventually I’m planning on letting you import 3D models into it and paint individual faces with colors from the palette.

Python

Bodge Script

Programmed in Python

Bodge Script is a programming language in it’s infancy. It is currently capable of parsing and lexing variable declaration, and setting, and it fully supports Algebraic expression trees. It is heavily inspired by GDscript, my favorite programming/scripting language.


The following is an example of runnable code:

  1. var width = 100
  2. var height = 25
  3. var perimeter = width*2+height*2
  4. var area = width * height
Output:

  1. EXEC DONE :
  2. Vars : {'width': 100.0, 'height': 25.0, 'perimeter': 250.0, 'area': 2500.0}
  3. Funcs : {}

TPG

Programmed in Python, using Tkinter and FFMPEG

TPG is a media file conversion program, It’s main strength is mass converting files/directories of files into another file type you can also change the output resolution.

GBPM

Programmed in Python, using Tkinter

GBPM is a program I made to help me create GUI programs in Python, because Tkinter is clunky by itself. You can attach function calls to certain events on each widget.
